# protein-ml-notebooks
Exploratory machine learning workflows for protein sequence representations

This repository contains small examples demonstrating:

- generation of protein embedding vectors
- dimensionality reduction of sequence representations
- visualization of sequence space

## Repository Structure

```
protein-ml-notebooks/
│
├ sequence_embedding_demo.py
├ embedding_pca_visualization.py
├ requirements.txt
│
└ data/
   └ example_embeddings.csv
```

## Installation

pip install -r requirements.txt

## Example Usage

Generate example embeddings:

python sequence_embedding_demo.py

Visualize embeddings:

python embedding_pca_visualization.py
